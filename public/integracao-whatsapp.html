Como conectar — Passo a passo
1
Escolha um provedor WhatsApp
Use Evolution API (open-source, gratuito) ou Z-API (plano gratuito disponível). Crie uma conta ou instale no servidor.
2
Escaneie o QR Code no painel do provedor
Gere a sessão no provedor e escaneie com o WhatsApp do número de teste (seu celular pessoal). Isso conecta o número ao provedor.
3
Configure o Webhook
No painel do provedor, localize Webhook / Integração e cole a URL acima. Configure o header X-Webhook-Secret com o valor da Secret Key acima.
4
Envie uma mensagem de teste
Envie uma mensagem para o número conectado. Se configurado corretamente, ela aparecerá no Log de Atividade acima e nas Conversas do CRM.
5
Valide o fluxo completo
✅ Mensagem recebida nas Conversas  ·  ✅ Scripts Comerciais inseríveis  ·  ✅ Mensagem editável  ·  ✅ Envio manual funciona  ·  ✅ Desconexão via painel do provedor
6
Trocar para número oficial
Após validação: desconecte o número de teste no provedor e escaneie o QR Code com o número oficial da PROSPEKT. O webhook URL e o secret permanecem iguais — zero mudança no CRM.
